Có 2 file để chạy là mua_1.command và mua_nhieu.command

------------------------------------------------------------------------------------
mua_1.command
Là để chạy mua 1 sản phẩm ( số lượng 1 ) trên 1 tài khoản duy nhất

Cách chạy:
B1: Mở file data_mua_1.json để sửa thông tin tài khoản, sản phẩm, cấu hình

có 3 mục cần lưu ý :
1: users: để sửa thông tin đăng nhập: thay đổi các thông tin username/ password
2: linkList ( sửa phần child ): thay thế link bằng link sản phẩm
3: phần config: sửa các thông tin giờ chạy
    hourRun: Giờ chạy
    minuteRun: Phút chạy
    secondRun: Giây chạy

B2: sau khi sửa xong file data_mua_1.json xong nhấp 2 lần liên tục vào mua_1.command để chạy tool


mua_nhieu.command
Là để chạy mua nhiều sản phẩm ( có thể thay đổi số lượng ) trên nhiều tài khoản
* Lưu ý mỗi tài khoản sẽ mua tất cả các sản phẩm bên dưới

------------------------------------------------------------------------------------
mua_nhieu.command
Cách chạy:
B1: Mở file data_mua_nhieu.json để sửa thông tin tài khoản, sản phẩm, cấu hình

có 3 mục cần lưu ý :
1: users: để sửa thông tin đăng nhập: thay đổi các thông tin username/ password có thể thêm nhiều user khác nhau

    Phần này lưu ý không để đặt trùng số tài khoản
        Cách thêm:
            Copy các thông tin username, pass, cvv. Sau đó thêm xuống dứoi, lưu ý cách phần tài khoản ban đầu bằng 1
            dấu phẩy
            VD:

            Trước khi thêm tài khoản                                Sau khi thêm tài khoản
            "users": [                                               "users": [
                {                                                       {
                  "username": "jrevexozip@gmail.com",                     "username": "jrevexozip@gmail.com",
                  "password": "Mrngoan0812",                              "password": "Mrngoan0812",
                  "cvv": "982"                                            "cvv": "982"
                }                                                       },
                ],                                                       {
                                                                          "username": "taikhoan2@gmail.com",
                                                                          "password": "Mrngoan0812",
                                                                          "cvv": "421"
                                                                        }
                                                                        ],

2: linkList ( sửa phần child ): thay thế link bằng link sản phẩm

    Phần này lưu ý không đặt trùng sản phẩm
        Cách thêm sản phẩm: Làm tương tự như trên tài khoản
            child: link sản phẩm
            quantity: số lượng muốn mua

    Trước khi thêm:

        "linkList": [
            {
              "child": "https://www.yodobashi.com/product/100000001003891482/"
            }
        ],

    Sau khi thêm:
     "linkList": [
        {
          "child": "https://www.yodobashi.com/product/100000001003891482/"
        },
        {
          "child": "https://www.yodobashi.com/product/138914814820320103/"
        }
     ],

3: phần config: sửa các thông tin giờ chạy
    hourRun: Giờ chạy
    minuteRun: Phút chạy
    secondRun: Giây chạy

B2: sau khi sửa xong file data_mua_nhieu.json xong nhấp 2 lần liên tục vào mua_nhieu.command để chạy tool
